QlikView to Qlik Sense Migration 

Objective:

To explore the possibility of converting QlikView dashboards into Qlik Sense using automation and AI.

Approach:

- QVW file is a binary format and cannot be directly parsed.

- Data extraction is simulated using CSV input.

- A Streamlit app dynamically generates dashboards.

- AI-based logic suggests chart types.

Features:

- Works with any CSV file

- Dynamic filtering

- KPI metrics

- Auto visualization generation

Limitations:

- Direct QVW to QVF conversion is not possible.

- Visualization extraction is simulated.

Conclusion:

Partial automation is feasible for data and visualization reconstruction.
